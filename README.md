# 🧥 Hoodie Shop Backend

A Spring Boot backend project for a simple e-commerce system focused on hoodies.

This project provides a REST API for managing products, users, carts, orders, and reviews.

---

## 🚀 Features

- Hoodie product management (CRUD)
- User management
- Shopping cart system
- Order processing (checkout & history)
- Product reviews & ratings
- RESTful API architecture
- Swagger API documentation

---

## 🛠️ Tech Stack

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- MariaDB 
- Maven
- Swagger (OpenAPI)

## 📦 Modules

### 👕 Products
- Add / update / delete / view hoodies

### 👤 Users
- Register and manage users

### 🛒 Cart
- Add/remove items from cart

### 📦 Orders
- Checkout and order history

### ⭐ Reviews
- Add and view product reviews
